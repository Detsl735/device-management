# Device Management API

Это монолитное приложение на NestJS для управления устройствами, сотрудниками, логами устройств и различными функциями компании, с интеграцией Telegram для авторизации.

## Основные возможности

* Управление устройствами: CRUD операции с устройствами, логирование событий устройств.
* Управление сотрудниками: CRUD операции с профилями сотрудников.
* Управление моделями, типами и статусами устройств.
* Авторизация и аутентификация пользователей через JWT.
* Интеграция с Telegram для авторизации и уведомлений.
* Разделение логики на модули (device, devicelog, employee, feature, model, profile, status, type, auth, telegram).
* Документация API с использованием Swagger.

## Структура проекта

```
src/
  auth/           # модуль аутентификации и авторизации
  device/         # модуль устройств, DTO, сущности и сервис
  devicelog/      # модуль логов устройств
  employee/       # модуль сотрудников
  feature/        # модуль функциональных возможностей устройств
  model/          # модуль моделей устройств
  profile/        # модуль профилей
  status/         # модуль статусов
  type/           # модуль типов устройств
  telegram/       # интеграция с Telegram (авторизация, уведомления)
```

## Технологии

* NestJS 10
* TypeScript
* PostgreSQL (через TypeORM)
* JWT для авторизации
* Telegram SDK для интеграции
* Swagger для документации API
* Jest для тестирования


## Интеграция с Telegram

Проект поддерживает авторизацию через Telegram InitData, что позволяет пользователям входить в систему только с использованием зарегестрированных в системе Telegram аккаунтов.
